# laravel-article-comment-track (aka lact)

# Local deployment
* required: `docker`, `docker compose`
* from project's directory:
* run `./app.sh start`
* run `./app.sh composer install`
* run `./app.sh stop` when done

# Usage
* go to http://localhost:8000

# Running tests
* run `./app.sh composer test`
